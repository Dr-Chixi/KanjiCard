# KanjiCard - Apprends les Kanjis 漢

KanjiCard est une application moderne pour apprendre les kanjis japonais en utilisant un système de répétition espacée (SRS) et des flashcards interactives.

## Fonctionnalités

- **Système SRS** : Apprentissage optimisé basé sur la science de la mémoire.
- **Support JLPT** : Decks organisés du niveau N5 au N4 (et plus à venir).
- **Decks Personnalisés** : Crée tes propres listes de kanjis à étudier.
- **Interface Moderne** : Design premium avec mode sombre et animations fluides.
- **Progression** : Gagne de l'XP, monte de niveau et suis tes statistiques.

## Installation Locale

1. Clone le dépôt.
2. Installe les dépendances :
   ```bash
   npm install
   ```
3. Configure tes variables d'environnement dans un fichier `.env` (voir `.env.example`).
4. Lance le serveur de développement :
   ```bash
   npm run dev
   ```

## Technologies

- **Frontend** : React, Vite, Tailwind CSS, Framer Motion.
- **Backend** : Supabase (Auth, Database, Edge Functions).
- **Design** : Shadcn/UI, Lucide Icons.
